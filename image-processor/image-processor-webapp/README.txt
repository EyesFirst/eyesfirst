You are almost certainly going to need to configure this web application for
your specific system.

The first step is to either install MATLAB or the MCR. Installers for these are
apparently not provided by MATLAB, so you will need to get one from an existing
install of MATLAB for the system you want to install the image processor on to.

Once those are installed, you will need to configure your servlet environment.
An example Tomcat Context.xml file is provided.
